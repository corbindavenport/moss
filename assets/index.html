About

Moss is a 6502 Microprocessor emulator, written in pure HTML/JavaScript. Moss can run assembly code written for the 6502 processor, and was named after the company that developed it, MOS Technology. It has full support for all 6502 assembly except inturrupts (meaning CLI, RTI and SEI instructions are unavailable).
